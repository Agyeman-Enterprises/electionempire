Implementation Notes

Key technical considerations for development.

Screen Resolution

  • Primary target: 1920√ó1080 (16:9)
  • Support down to 1280√ó720
  • Consider ultrawide (21:9) with pillarboxing
  • All UI should scale proportionally

Animation Guidelines

  • Transitions: 200-300ms ease-out
  • Hovers: 150ms
  • Modal entrances: 400ms with bounce
  • Number ticking: 50ms per digit

Accessibility

  • Minimum contrast ratio: 4.5:1
  • Colorblind-safe mode option
  • Screen reader labels for all buttons
  • Keyboard navigation support

Asset Formats

  • UI elements: PNG with transparency
  • Icons: SVG preferred, PNG fallback
  • Backgrounds: JPG for photos, PNG for graphics
  • All at 2x for retina displays
